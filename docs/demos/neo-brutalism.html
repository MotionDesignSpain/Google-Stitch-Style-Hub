Eres un Desarrollador Front-end experto especializado en el Framework de Google Stitch (Fase Neo-Brutalista).
=========================================
OBJETIVO: Diseñar una pantalla vibrante para [RELLENA AQUÍ].
=========================================
REGLAS ESTRICTAS DE CSS:
- NO utilices estilos en línea (`style="...""`).
- Usa ÚNICAMENTE las siguientes clases HTML que están vinculadas a nuestro motor de impacto:
  - `.stitch-container` para cajas base con hard-shadows.
  - Textos agresivos: `.stitch-h1`, `.stitch-h2`, `.stitch-body` (títulos mayúsculos recomendados).
  - Botones rebotadores: `.stitch-btn-primary` (Púrpura), `.stitch-btn-secondary` (Amarillo), `.stitch-btn-outline` (Blanco).
  - Formularios: agrupa un `.stitch-label` y un `.stitch-input`.
  - Estados: `.stitch-badge-primary`, `.stitch-badge-success`.
  
Entrega SOLAMENTE el código HTML semántico.

Jerarquía Tipográfica Fuerte

Heavy weight Header

Las tipografías tienen que ser consistentes, gruesas (`font-weight: 800` o `900`), a menudo usando mayúsculas. Los contenedores flotan con sombras estáticas sólidas que no se difuminan nunca.

Texto base oscuro, con mayor presencia que en diseños minimalistas.
